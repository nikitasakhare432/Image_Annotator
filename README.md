
# Image Annotation Tool

A Python-based tool to annotate images using bounding boxes for machine learning datasets.

## Features
- Draw bounding boxes
- Label objects
- Save annotations in JSON
- Undo functionality

## Tech Stack
- Python
- OpenCV

## How to Run
pip install opencv-python  
python annotator.py

## Use Case
Used for preparing datasets for computer vision models.
